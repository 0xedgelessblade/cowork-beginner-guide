# Cowork Beginner Guide — 新手設定指南

一份給 Cowork 新手的完整工作環境設定教學。

## 這是什麼？

這個 repo 收錄了從零開始設定 Claude Cowork 工作環境的 step-by-step 指南，包含：

- 資料夾結構建立
- 記憶檔（about-me、lessons-learned）設定
- HTML 輸出偏好設定
- CLAUDE.md 工作守則撰寫

## 適合誰？

剛開始使用 Claude Cowork，想要有一個結構化工作環境的使用者。

## 使用方式

把指南內容直接貼給 Claude Cowork，它會按照步驟幫你設定好一切。
